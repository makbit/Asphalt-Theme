## Installation via VS Code

1. Open **Extensions** sidebar panel in VS Code. `View  Extensions`
2. Search for `MakBit Asphalt Theme`
3. Click **Install** to install it
4. Code > Preferences > Color Theme > **MakBit Asphalt Theme**
